# HiiotUI 组件库

欢迎使用 **HiiotUI** - 一个现代化的 React 组件库！

## 🚀 特性

- **现代化设计** - 基于最新的设计趋势和用户体验最佳实践
- **TypeScript 支持** - 完整的类型定义，提供更好的开发体验
- **Tailwind CSS** - 使用 Tailwind CSS 构建，支持主题定制
- **响应式设计** - 所有组件都支持响应式布局
- **可访问性** - 遵循 WCAG 2.1 可访问性标准
- **Tree Shaking** - 支持按需导入，优化打包体积

## 📦 安装

```bash
npm install hiiot-ui
# 或
yarn add hiiot-ui
# 或
pnpm add hiiot-ui
```

## 🎯 快速开始

```tsx
import { Button, Input } from 'hiiot-ui'
import 'hiiot-ui/dist/index.css'

function App() {
  return (
    <div>
      <Button variant="primary">点击我</Button>
      <Input label="用户名" placeholder="请输入用户名" />
    </div>
  )
}
```

## 🎨 组件

### 基础组件

- **Button** - 按钮组件，支持多种变体和状态
- **Input** - 输入框组件，支持标签、错误提示和帮助文本

### 即将推出

- **Card** - 卡片组件
- **Modal** - 模态框组件
- **Layout** - 布局组件
- 更多组件正在开发中...

## 🛠️ 开发

本组件库使用以下技术栈：

- **React 18** - 用户界面库
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Rollup** - 打包工具
- **Storybook** - 组件文档和开发环境
- **Vite** - 开发服务器
